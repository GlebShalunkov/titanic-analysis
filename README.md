# 🚢 Titanic Survival Analysis

**Проект:** Разведочный анализ (EDA) и интерактивный дашборд для датасета Титаник

**Автор:** GlebShalunkov  
**Технологии:** Python, Pandas, Matplotlib, Seaborn, Streamlit

---

## 📊 О проекте
Классический анализ данных пассажиров Титаника.  
Цель: выявить ключевые факторы выживаемости и представить результаты в интерактивном дашборде.

## 🎯 Что сделано
- Обработка пропусков (возраст, порт посадки)
- Создание новых признаков (титулы из имён, размер семьи)
- Анализ выживаемости по полу, классу, возрасту
- Построение 6+ визуализаций
- Интерактивный дашборд на Streamlit

## 🔥 Ключевые инсайты
- Женщины выживали в 3.9 раза чаще мужчин
- Пассажиры 1 класса — 63% выживаемости, 3 класса — 24%
- Дети до 18 лет выживали чаще (54% против 36%)

## 📁 Файлы

- `app.py` — Streamlit дашборд
